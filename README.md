- `test.js` → Test cases

##  How to Run
1. Clone the repository and open the folder:
   ```bash
   git clone <your-repo-url>
   cd weekday-aggregation-js
   
2. Run the tests using Node:

   ```bash
      node test.js

see console outputs showing the results for each test case.
